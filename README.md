# Headwater Institute

The official website for the Headwater Institute — a premium leadership development
company teaching the Agency Model through wilderness expeditions, executive coaching,
marriage retreats, and athletic performance programs.

**Live site:** https://headwaterinstitute.com
**Foundation:** https://headwatergives.org

---

## Site Structure

```
/                  → Home page (index.html)
/store             → The Reading Room — curated book canon (store/index.html)
/model             → The Agency Model — interactive Agency Wheel,
                     canon deep-dives, leadership cluster map (model/index.html)
/404.html          → Custom not-found page
netlify.toml       → Hosting configuration (headers, redirects)
```

## Tech Stack

Plain HTML, CSS, and JavaScript. No build step, no framework, no dependencies.
Each page is a single self-contained file. Hosted on Netlify, deployed automatically
from this repository on every push to `main`.

## Brand

- **Primary palette:** Parchment (#F7F4EF), Ink (#0D0F14), Gold (#C8A86E), Sage (#7B9E87)
- **Typography:** Cormorant Garamond (display), Inter (body), Space Mono (labels/data)
- **Founder:** Barrett Christensen

## Updating the Site

1. Edit the relevant HTML file locally or directly in GitHub
2. Commit the change
3. Netlify automatically rebuilds and deploys within ~30 seconds — no manual steps needed
